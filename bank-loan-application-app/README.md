# Bank Loan Application App

This folder is separated from `bug_daddy` and contains the bank loan application pieces:

- `frontend/` - Next.js application
- `backend/api/` - FastAPI backend
- `backend/microservices/` - loan workflow microservices/Lambda handlers

## Run locally

From this folder:

```bash
./local_run.sh
```

Frontend runs at `http://localhost:3000` and backend runs at `http://localhost:8000`.
